# 💪 FitLock — Sweat to Scroll

Turn screen time addiction into fitness gains. Do pushups to unlock your apps.

## 🚀 Live Demo
**https://fitlock-five.vercel.app**

## 📁 Project Structure

```
fitlock/
├── index.html              # Live web prototype
├── docs/
│   └── roadmap.md          # Development roadmap
├── ios-native/
│   └── FitLock.swift       # Native iOS (SwiftUI) code
└── mobile-app/
    └── expo-code.js        # Expo + React Native code
```

## 🎯 What's Included

### 1. Web Prototype (`index.html`)
- Fully working HTML/JS app
- Real AI pose detection with TensorFlow.js
- Runs in any modern browser
- Deployed live on Vercel

### 2. iOS Native Code (`ios-native/`)
- Complete SwiftUI code
- Uses Apple Vision Framework for pose detection
- Ready to open in Xcode

### 3. Mobile App Code (`mobile-app/`)
- Expo + React Native
- Cross-platform (iOS + Android)
- Uses TensorFlow.js for pose detection

### 4. Documentation (`docs/`)
- Full development roadmap
- 12-week plan to App Store

## 🛠️ Tech Stack

- **Web:** HTML, Tailwind CSS, React (via CDN), TensorFlow.js
- **iOS Native:** Swift, SwiftUI, Vision Framework
- **Mobile App:** Expo, React Native, JavaScript

## 📱 Features

- ✅ Exercise-based app unlocking
- ✅ AI pose detection (pushup counting)
- ✅ Streak tracking
- ✅ Squad/social leaderboards
- ✅ Stats & analytics
- ✅ Profile customization
- ✅ Settings & preferences
